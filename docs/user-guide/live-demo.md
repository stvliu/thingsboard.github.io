实时演示是一个单节点服务器，用于简化 ThingsBoard 平台的入门过程。

您可以使用以下 URL 访问演示服务器 [**demo.thingsboard.io**](https://demo.thingsboard.io/signup)

您需要填写注册表格，然后您将收到您专用租户的租户管理员帐户。

登录后，您将可以访问 ThingsBoard [**示例**](/docs/samples/) 中使用的所有仪表板和设备。
