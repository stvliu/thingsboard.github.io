[NodeMCU](https://en.wikipedia.org/wiki/NodeMCU) 是一个开源 IoT 平台。它包括在 ESP8266 Wi-Fi SoC 上运行的固件和基于 ESP-12 模块的硬件。

在 NodeMCU 上运行的这些示例中的应用程序使用 Lua 脚本语言编写，它非常简单且易于理解。
