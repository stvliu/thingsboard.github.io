[Arduino](https://en.wikipedia.org/wiki/Arduino) 是一个开源原型平台，基于易于使用的硬件和软件。
Arduino 板能够读取来自传感器或按钮的输入，对其进行处理并将其转换为输出。在 Arduino 上运行的这些示例中的应用程序是使用 [Arduino IDE](https://www.arduino.cc/en/Main/Software) 开发的。
