[ESP8266](https://en.wikipedia.org/wiki/ESP8266) 是一款非常流行的低成本 Wi-Fi 芯片，支持 TCP/IP 协议栈和 MCU 功能。
本示例中的应用程序在 ESP8266 内核上运行，适用于 Arduino，并使用 [Arduino IDE](https://www.arduino.cc/en/Main/Software) 开发。
