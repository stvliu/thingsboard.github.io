GridLinks 允许您将数据从窗口小部件导出到 CSV 或 XLS。如果适用于窗口小部件类型，几乎任何窗口小部件都支持此功能。
导出数据集中的列列表对应于您在数据集配置中配置的参数列表。

请参阅下面的视频教程，了解如何使用此功能的分步说明。
