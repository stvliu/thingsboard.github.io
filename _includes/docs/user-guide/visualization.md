ThingsBoard 允许您配置可自定义的物联网仪表板。
每个物联网仪表板可以包含多个小部件，这些小部件可视化来自多个物联网设备的数据。
创建物联网仪表板后，您可以将其分配给物联网项目的多个客户。

物联网仪表板重量轻，您可以拥有数百万个仪表板。
